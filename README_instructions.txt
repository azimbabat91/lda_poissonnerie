INSTRUCTIONS POUR LANCER LE SITE LDA POISSONNERIE

1. Ouvre ce dossier dans ton explorateur de fichiers.
2. Clique-droit dans un espace vide > "Ouvrir dans le terminal" (ou invite de commande).
3. Tape les commandes suivantes :

   python -m venv venv
   venv\Scripts\activate   (ou 'source venv/bin/activate' sur Mac/Linux)
   pip install -r requirements.txt
   python app.py

4. Ouvre ton navigateur sur http://127.0.0.1:5000
